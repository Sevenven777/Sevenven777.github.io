我这提示了一个错误
Did you install mysqlclient or MySQL-python?
安装了MySQL-python后依旧提示这个错误

搜了搜解决办法：
django 搭建mysql数据库全过程

python2.x使用MySQLdb，python3.x使用PyMysql模块。
如果出现这个，我们首先pip install pymysql, 我用的是python的2.13 版本
然后在django项目的init下设置：
import pymysql
pymysql.install_as_MySQLdb()


再运行一下就没有问题了

说明配置成功了~
